Project Structure:
This project is organized into two main parts:
1. Back-end (Back/): Handles camera systems, YOLO/face recognition models, 
   database operations, and log management.
2. Front-end (Front/): Provides the web dashboard with live streams, charts, 
   and face recognition logs.

Databases are divided into:
- crowd_management_database/: Manages crowd data, measurements, and face recognition.
- camera_system_database/: Manages camera system data and configurations.

requirements.txt specifies Python dependencies.
